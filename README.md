## The workshop series

Three one-hour sessions that take you from your first day with a Copilot licence to confident, everyday use. The series is mandatory and sequential, so the most value comes from attending in order.

- **101 · Copilot Chat: Your Foundation** — safety, the interface, custom instructions, the everyday wins, and a quick data spring clean.
- **102 · PowerPoint, Excel and Word: Create and Build** — working from your own content, Edit with Copilot, Plan with Copilot, and a choice of model.
- **103 · Teams, Outlook, Facilitator, Researcher and Analyst** — meetings, your inbox, your time, and the three ready-made agents.

Each session is delivered as a live demonstration from a safe demo account, is recorded and shared, and comes with a walkthrough document to keep.

## For maintainers

This repository is a static site served by **GitHub Pages** from the `main` branch (root folder). The landing page is `index.html`; everything else is a self-contained HTML file linked from it.

**To add or update a resource:**

1. Upload the HTML file to the repository (**Add file → Upload files → Commit to `main`**). GitHub Pages redeploys automatically within a minute or so.
2. Open `index.html` and add or update the matching card so the new file is linked from the hub.
3. Commit, then hard-refresh the live site to confirm.

**Good to know:**

- **Filenames with spaces** must be URL-encoded in links (a space becomes `%20`). For example, `Digital Heroes - Welcome Guide.html` is linked as `Digital%20Heroes%20-%20Welcome%20Guide.html`. The cards in `index.html` already follow this. If you rename a file, update its link to match.
- **Self-contained files.** Each page embeds its own images, fonts and scripts, so there are no broken asset links and files can be moved or shared individually.
- **Branding.** The landing page uses the core Digital Heroes brand palette (navy, blue and gold). The workshop decks use the teal Workshops & Training category colour. Both are intentional.
